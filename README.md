# Analisa Sentimen Stockbit

Repositori ini berisi analisis sentimen pada ulasan aplikasi Stockbit.

File penting:
- `Analisa_Sentimen_Stockbit.ipynb` — notebook analisis
- `scraping_data.ipynb` — notebook scraping data
- `Stockbit_ulasan_aplikasi.csv` — dataset ulasan

Instruksi singkat:
1. Buat virtualenv: `python -m venv .venv`
2. Aktifkan: `source .venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Jalankan notebook di Jupyter.
